Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a blueprint that allows developers to access and modify the elements of a web page. JavaScript makes a webpage interactive by manipulating the DOM

How do you select an element from the DOM?

For single element: document.getElementById() and document.querySelector()
For multiple element: document.getElementByClass() ,document.getElementByTagName(),document.querySelectorAll()

What is event delegation in the context of the DOM, and why it is useful?

Event delegation allows a parent element to handle events from its child elements, improving efficiency in the DOM.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's using setAttribute() , getAttribute() , removeAttribute() , and can styles add .className and classList[add(),remove()]
